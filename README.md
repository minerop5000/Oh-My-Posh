## setup

- C:\Users\...\Documents\WindowsPowerShell\Microsoft.PowerShell_profile.ps1
  
```oh-my-posh --init --shell pwsh --config "C:\git\Oh-My-Posh\v1.omp.json" | Invoke-Expression```
